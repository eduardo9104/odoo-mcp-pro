How to Create an Odoo API Key

Each user needs their own API key. The key determines what data you can access in Odoo — your existing Odoo permissions apply.

  1. 1

    Log in to your Odoo instance

    {% if user_tenant %} Go to {{ user_tenant.odoo_url }} {% else %} Go to your Odoo URL (e.g. https://yourcompany.odoo.com) {% endif %}

  2. 2

    Open your profile

    Click your avatar (top-right) → My Profile, or go to Settings → Users → click your name.

  3. 3

    Go to the Account Security tab

    Scroll down or click the Account Security tab on your profile page.

  4. 4

    Create a new API Key

    Click New API Key, enter a description (e.g. "Claude MCP"), and confirm with your password. Copy the key immediately — it will only be shown once.

Important: Your API key has the same permissions as your Odoo account. Never share it with others. Each team member should create their own key.

Your Odoo connection is ready. Choose how you want to use it with Claude:

Claude.ai (browser)

Claude.ai connects automatically via OAuth. No extra setup needed.

  1. Go to Claude.ai and open a new conversation
  2. Click the MCP integrations icon (puzzle piece) in the chat bar
  3. Search for "Odoo" and click Connect
  4. Sign in with your same account — you'll be connected automatically

Claude Code (CLI / VS Code)

Add the Odoo MCP server to your Claude Code configuration:

{
  "mcpServers": {
    "odoo": {
      "url": "{{ mcp_server_url }}/mcp/"
    }
  }
}

Add this to ~/.claude/settings.json or your project's .mcp.json. You'll be prompted to sign in on first use.

Claude Desktop

Add the server to your Claude Desktop configuration file:

{
  "mcpServers": {
    "odoo": {
      "url": "{{ mcp_server_url }}/mcp/"
    }
  }
}

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json
