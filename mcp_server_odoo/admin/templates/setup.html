How your API key is protected

  • Your API key is stored securely and only accessible by the server.
  • The key is only used server-side to authenticate requests to your Odoo instance on your behalf.
  • Odoo enforces permissions -- your API key determines what Claude can see and do, just like when you use Odoo directly.
  • You can revoke your API key in Odoo at any time, instantly cutting off access. You can also remove your connection above.
