# Odoo MCP v1.2.0 — Teams, Dashboard & Better Limits

Hi everyone,

We've shipped a big update to Odoo MCP. Here's what's new:

## What's new

**Bigger results, faster workflow**
Search now returns up to 100 records by default (was 10). No more "show me the next page" back-and-forth.

**Teams**
Everyone connected to the same Odoo instance is now part of a team. The first person who connected becomes the team admin and can invite colleagues with a shareable link. Each team member uses their own API key with their own Odoo permissions.

**Admin dashboard**
Team admins can see who's on their team, their connection status, and usage stats.

**Improved setup instructions**
The setup page now walks you through the full Connect flow step by step — including the OAuth popup and sign-in.

**Zero-downtime deploys**
We've upgraded our infrastructure so future updates won't interrupt your connection anymore.

## Action needed: please reconnect

To pick up these changes, please **disconnect and reconnect** your Odoo MCP connector in Claude:

1. Go to Claude → MCP connectors
2. Click the **...** menu on Odoo → **Disconnect**
3. Click **Connect** again and sign in

That's it — takes 10 seconds.
